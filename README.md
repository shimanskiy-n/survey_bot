# Простой Telegram-бот на Python (aiogram + SQLite)

Этот проект — минимальный пример Telegram-бота на `aiogram`, который:

- **При старте (`/start`)** спрашивает у пользователя его **ФИО** и сохраняет его в базе данных SQLite, привязывая к его Telegram ID.
- После успешной регистрации задаёт пользователю **вопрос** и сохраняет его **ответ** в той же базе.

### Стек

- **Язык**: Python 3.10+
- **Библиотека для бота**: `aiogram 3`
- **База данных**: SQLite

---

### Структура проекта

- **`app/`** — исходный код бота  
  - **`__init__.py`** — делает папку модулем Python  
  - **`config.py`** — загрузка настроек (токен бота)  
  - **`db.py`** — работа с базой данных SQLite  
  - **`handlers.py`** — обработчики команд и сообщений  
  - **`main.py`** — точка входа, запуск бота  
- **`requirements.txt`** — зависимости проекта  
- **`README.md`** — документация проекта  

---

### Установка

1. **Клонировать / скопировать проект** (или просто создать папку и файлы как здесь).

2. **Создать и активировать виртуальное окружение (рекомендуется)**:

```bash
python -m venv .venv
.venv\Scripts\activate
```

3. **Установить зависимости**:

```bash
pip install -r requirements.txt
```

4. **Создать Telegram-бота и получить токен** у [BotFather](`https://t.me/BotFather`).

5. **Указать токен бота** через переменную окружения:

```bash
set BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
```

На Windows PowerShell:

```powershell
$env:BOT_TOKEN="ВАШ_ТОКЕН_ОТ_BOTFATHER"
```

> В коде токен читается из переменной окружения `BOT_TOKEN`. Файл `.env` вы можете использовать сами, но он уже добавлен в `.gitignore`, чтобы не попасть в репозиторий.

### Структура данных в базе (SQLite)

Используется один файл базы данных `bot.db` в корне проекта.

Таблица `users`:

- `id` — целочисленный первичный ключ
- `telegram_id` — Telegram ID пользователя (уникальный)
- `full_name` — строка с ФИО
- `question` — вопрос, который бот задал пользователю
- `answer` — ответ пользователя

Таблица создаётся автоматически при старте бота.

---

### Запуск бота

Из корня проекта:

```bash
python -m app.main
```

После запуска:

1. Откройте вашего бота в Telegram.
2. Напишите ему команду `/start`.
3. Бот попросит ввести **ФИО** (одной строкой).
4. После сохранения ФИО бот задаст вопрос.
5. Вы отправите ответ, и он сохранится в базе.

---

### Где доработать под себя

- **Сменить вопрос** — файл `handlers.py`, константа `BOT_QUESTION`.
- **Добавить новые поля в БД** — файл `db.py` (структура таблицы) и `handlers.py` (логика сохранения).
- **Изменить язык сообщений** — всё текстовое общение с пользователем находится в `handlers.py`.
